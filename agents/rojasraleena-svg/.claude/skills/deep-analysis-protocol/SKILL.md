# Deep Analysis Protocol

用于复杂评审任务的标准流程。

## 何时使用

- 需要高可信度结论
- 存在争议性或高影响决策
- 需要多来源证据支撑

## 标准流程

1. Research: 收集证据与来源链接。
2. Synthesis: 形成多个候选结论。
3. Critique: 识别漏洞与缺证。
4. Verify: 核验来源可追溯性。
5. Decide: 输出最终结论与不确定性。

## 质量门槛

- 每个关键结论至少 1 个可追溯 URL。
- 不能验证的断言必须标注“缺证据/不确定”。
- 最终结论必须包含 Sources 与 Uncertainties。
