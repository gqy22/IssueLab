# lean4代码检查与逻辑梳理

## 输入期望

- lean4代码片段或文件路径
- 期望的定理陈述与上下文
- 已知报错或失败信息

- 识别类型不匹配与语义不一致

## 输出格式

- 问题定位
- 证明梳理
- 修正建议
